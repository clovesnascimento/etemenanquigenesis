### Projeto principal

**[Etemenanqui](https://github.com/clovesnascimento/etemenanqui)** — Linguagem artificial de protocolo para compressão de prompts em sistemas de IA. Reduz custo computacional de prompts técnicos em ~26% via arquitetura morfológica CVC (Modelo B), eliminando redundância estrutural sem perda semântica.

| Métrica | Resultado |
|---|---|
| BPE ratio | **0.742** (±15%) |
| gz bytes | **−50%** |
| H_char | **−20%** |
| Diagnóstico | ✓✓✓ todas as camadas |
